# ZegGame
Tytuł - Gra przeglądarkowa Labirynt. Opis - Cel projektu to stworzenie nowoczesnej, przeglądarkowej gry typu labirynt, gdzie gracz steruje postacią poruszającą się po planszy, zbiera przedmioty, unika zagrożeń oraz rozwiązuje zagadki na kolejnych, coraz trudniejszych etapach. Czas realizacji: od 10.04.2026-8.06.2026. Autor: Arkadiusz Grzybek
